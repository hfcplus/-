# 前端传list

##### 前端代码：

```javascript
function setDatDocumentRight(){
  var roleIds = [];
  roleIds.push("1");
  roleIds.push("2");
  roleIds.push("3");
  $.ajax({
    url: '',
    type: "post",
    data: {
      roleIds: roleIds
    },
    success:function(resp){
        alert(resp);
    }
  });
}
```

##### 后端代码：

```java
@RequestMapping("/ajaxList")
@ResponseBody
public void testAjaxList(@RequestParam("roleIds[]") List<String> roleIds) {
    for (String roleId : roleIds) {
        System.out.println(roleId);
    }
}
```
